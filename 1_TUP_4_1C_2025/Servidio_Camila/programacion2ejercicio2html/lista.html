Tipos de Listas en HTML

Lista Ordenada

Las listas ordenadas muestran los elementos en un orden específico.

  1. Primer paso
  2. Segundo paso
  3. Tercer paso

Lista Desordenada

Las listas desordenadas muestran los elementos sin un orden específico.

  • Manzana
  • Banana
  • Naranja

Lista de Definición

Las listas de definición se usan para términos y sus definiciones.

HTML
Lenguaje de marcado para la creación de páginas web.
CSS
Lenguaje de estilos para el diseño de páginas web.

Lista Dependiente (Anidada)

Las listas dependientes contienen sublistas dentro de los elementos.

  • Frutas
    • Manzana
    • Banana
    • Naranja
  • Verduras
    • Lechuga
    • Tomate
    • Zanahoria
